## PDF 智能分析系统（前端阶段）

本项目是一个面向 PDF 文档的智能分析系统，目标是通过 **OCR、文本摘要生成以及知识图谱构建** 等技术，为用户提供对 PDF 内容的高效理解与结构化分析。

该目录下的文件是 **前端部分**，后端功能（OCR / 摘要生成 / 知识图谱生成）请在根目录/backend下查看。

---

## 一、当前完成进度（截至目前）

### ✅ 已完成内容（前端）

#### 1. Web 前端整体框架
- 采用 **HTML + CSS + JavaScript** 实现
- 页面结构清晰，分为：
  - 顶部系统标题栏
  - 文件上传与文件信息展示栏
  - 左侧 PDF 展示 Panel
  - 右侧分析 Panel（摘要 / 思维导图）

#### 2. PDF 浏览功能（基于 PDF.js）
已实现完整的 PDF 浏览能力，包括：
- 本地 PDF 文件上传
- PDF 页面渲染显示
- 上一页 / 下一页翻页
- PDF 缩放（放大 / 缩小）
- 固定 PDF Panel 尺寸，缩放仅影响内容大小
- 缩放后支持在 Panel 内滚动查看页面内容

#### 3. 前端交互与布局
- 左右双栏布局（PDF Panel + 分析 Panel）
- 右侧分析区支持 Tab 切换：
  - **输出（摘要）**
  - **思维导图**
- Tab 为定宽样式，符合工具型产品交互习惯
- 根据当前 Tab 显示对应的主操作按钮：
  - 输出 → 生成摘要
  - 思维导图 → 生成知识图谱
- 当前按钮仅为 UI 占位，尚未接入实际逻辑
